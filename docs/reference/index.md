# Package index

## Functions

These are the main functions for working with the ISCAM package.

- [`iscamaddexp()`](https://iscam4.github.io/ISCAM/reference/iscamaddexp.md)
  : Overlay an Exponential Density Function on Histogram
- [`iscamaddlnorm()`](https://iscam4.github.io/ISCAM/reference/iscamaddlnorm.md)
  : Overlay a Log Normal Density Function on Histogram
- [`iscamaddnorm()`](https://iscam4.github.io/ISCAM/reference/iscamaddnorm.md)
  : Overlay a Normal Density Function on Histogram
- [`iscamaddt()`](https://iscam4.github.io/ISCAM/reference/iscamaddt.md)
  : Overlay a t Density Function on Histogram
- [`iscamaddtnorm()`](https://iscam4.github.io/ISCAM/reference/iscamaddtnorm.md)
  : Overlay a t Density Function and a Normal Density Function on
  Histogram
- [`iscambinomnorm()`](https://iscam4.github.io/ISCAM/reference/iscambinomnorm.md)
  : Overlays Normal Approximation onto Binomial
- [`iscambinompower()`](https://iscam4.github.io/ISCAM/reference/iscambinompower.md)
  : Rejection Region for Binomial
- [`iscambinomprob()`](https://iscam4.github.io/ISCAM/reference/iscambinomprob.md)
  : Calculate Binomial Tail Probabilities
- [`iscambinomtest()`](https://iscam4.github.io/ISCAM/reference/iscambinomtest.md)
  : Exact Binomial Test
- [`iscamboxplot()`](https://iscam4.github.io/ISCAM/reference/iscamboxplot.md)
  : A box plot
- [`iscamchisqprob()`](https://iscam4.github.io/ISCAM/reference/iscamchisqprob.md)
  : Chi-Square Probability
- [`iscamdotplot()`](https://iscam4.github.io/ISCAM/reference/iscamdotplot.md)
  : A dot plot
- [`iscamhypernorm()`](https://iscam4.github.io/ISCAM/reference/iscamhypernorm.md)
  : Hypergeometric p-value and Distribution Overlaid with Normal
  Distribution
- [`iscamhyperprob()`](https://iscam4.github.io/ISCAM/reference/iscamhyperprob.md)
  : Hypergeometric p-value and Distribution
- [`iscaminvbinom()`](https://iscam4.github.io/ISCAM/reference/iscaminvbinom.md)
  : Inverse Binomial Probability
- [`iscaminvnorm()`](https://iscam4.github.io/ISCAM/reference/iscaminvnorm.md)
  : Inverse Normal Calculation
- [`iscaminvt()`](https://iscam4.github.io/ISCAM/reference/iscaminvt.md)
  : Inverse T Calculation
- [`iscamnormpower()`](https://iscam4.github.io/ISCAM/reference/iscamnormpower.md)
  : Rejection Region for Normal
- [`iscamnormprob()`](https://iscam4.github.io/ISCAM/reference/iscamnormprob.md)
  : Normal Tail Probability
- [`iscamonepropztest()`](https://iscam4.github.io/ISCAM/reference/iscamonepropztest.md)
  : One Proportion Z-Test and Interval
- [`iscamonesamplet()`](https://iscam4.github.io/ISCAM/reference/iscamonesamplet.md)
  : One Sample T-Test
- [`iscamsummary()`](https://iscam4.github.io/ISCAM/reference/iscamsummary.md)
  : Some Summary Statistics
- [`iscamtprob()`](https://iscam4.github.io/ISCAM/reference/iscamtprob.md)
  : Tail Probability for t-distribution
- [`iscamtwopropztest()`](https://iscam4.github.io/ISCAM/reference/iscamtwopropztest.md)
  : Two Proportion Z-Test and Interval
- [`iscamtwosamplet()`](https://iscam4.github.io/ISCAM/reference/iscamtwosamplet.md)
  : Two Sample T-Test
